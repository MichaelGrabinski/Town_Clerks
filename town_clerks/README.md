Town of East Hartford - Town Clerks Office

Starter Django app skeleton for Town Clerks office data ingest and viewing.

Contents:
- clerks: main Django app for clerk data
- manage.py: Django management wrapper
- requirements.txt: Python deps

Next steps:
1. Configure your .env with DB credentials and ADFS settings.
2. pip install -r requirements.txt
3. python manage.py migrate
4. python manage.py runserver
